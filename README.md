# ❌⭕ Tic-Tac-Toe AI with GUI (Uniform Cost Search)

## 📌 Overview
This project is a Tic-Tac-Toe game built using **Python** and **Tkinter**, featuring an AI opponent that uses **Uniform Cost Search (UCS)** to determine the best move.

---

## 🎯 Features
- Interactive 3×3 Tic-Tac-Toe GUI  
- Human vs AI gameplay  
- AI decision-making using search algorithm  
- Automatic win/tie detection  
- Game reset after completion  

---

## 🧠 Approach

- Implemented game logic for move validation and winner detection  
- Designed an AI agent using **Uniform Cost Search (UCS)**  
- The AI evaluates possible board states and selects the optimal move  

---

## 🛠️ Technologies Used
- Python  
- Tkinter (GUI)  
- Heap Queue (Priority Queue for UCS)  

---

## ⚙️ How It Works
1. Player makes a move on the board  
2. The AI evaluates possible moves using UCS  
3. The best move is selected based on cost  
4. The game continues until:
   - A player wins  
   - Or a tie occurs
